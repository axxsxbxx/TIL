# 리스트(list)

## 1. 정의

> 저장 가능한 데이터의 타입, 데이터 셋의 종류에 제한이 없음
>
> 서로 다른 구조의 데이터를 하나로 묶을 수 있는 자료구조

- R에서는 통계 분석 결과가 리스트 구조로 제시되는 경우가 많음



## 2. 생성 방법

```R
# list() 함수 이용
> l<-list(1,2,3)
> l
[[1]]
[1] 1

[[2]]
[1] 2

[[3]]
[1] 3


# 서로 다른 다양한 유형의 객체를 동시에 저장 가능
# 인덱스 이름을 생성할 때 정할 수 있음
> a<-list(a = 1:3, b = "a string", c = pi, d = list(-1,-5))
> a
$a
[1] 1 2 3

$b
[1] "a string"

$c
[1] 3.141593

$d
$d[[1]]
[1] -1

$d[[2]]
[1] -5

# 인덱스 이름을 names() 함수를 통해 추출할 수 있음
> names(a)
[1] "a" "b" "c" "d"

```



## 3. 인덱싱 및 주요 기능

```R
> a<-list(a = 1:3, b = "a string", c = pi, d = list(-1,-5))
> a
$a
[1] 1 2 3

$b
[1] "a string"

$c
[1] 3.141593

$d
$d[[1]]
[1] -1

$d[[2]]
[1] -5

# list[n]
# 리스트의 n번째 원소가 리스트 형식으로 추출됨. 리스트로부터 추출된 서브리스트
> a[1]
$a
[1] 1 2 3
> a[1]+1
Error in a[1] + 1 : 이항연산자에 수치가 아닌 인수입니다

# list[[n]]
# 리스트의 n번째 원소가 선택됨. 원소 자체가 가지고 있는 형식을 그대로 유지한 채 추출됨
> a[[1]] # a[["a"]]
[1] 1 2 3
> a[[1]][1]
[1] 1
> a[[1]]+1
[1] 2 3 4

# list$index_name
# 리스트에서 주어진 이름을 가지는 인덱스의 원소가 선택됨
# 원소 자체가 가지고 있는 형식을 그대로 유지한 채 추출됨
> a$a
[1] 1 2 3
> a$a[1]
[1] 1

# NULL
# 리스트로부터 특정 원소를 제거하기 위해서 리스트 인덱싱을 이용하여 원소를 선택한 후 NULL을 할당함
a[["b"]] <- NULL

> names(a)
[1] "a" "b" "c" "d"
> names(a) <- NULL
> names(a)
NULL

# unlist()
# 리스트를 해제함. 리스트를 벡터로 반환함
> a[1]
[[1]]
[1] 1 2 3
> new_a <- unlist(a[1])
> new_a
[1] 1 2 3

```
